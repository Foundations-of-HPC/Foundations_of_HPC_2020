Save configuration
==================

**Syntax**:(savecfg)

Save current configuration to ~/.xaosrc (under Unix) or xaos.cfg (under DOS and
Windows). XaoS automatically reloads the configuration from this file when it
starts.

**Available as**: menu item, command line option, command
